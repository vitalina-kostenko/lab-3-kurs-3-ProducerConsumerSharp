# ProducerConsumerSharp

Лабораторна робота — реалізація задачі «Виробник–Споживач» з використанням семафорів на C#.

## Опис

Програма реалізує класичну задачу синхронізації «Виробник–Споживач» з довільною кількістю виробників і споживачів, які взаємодіють через сховище з обмеженою місткістю.

## Засоби синхронізації

- **`Semaphore full`** (лічильний семафор, init = capacity) — кількість вільних місць у сховищі
- **`Semaphore empty`** (лічильний семафор, init = 0) — кількість елементів у сховищі
- **`Semaphore access`** (бінарний семафор, init = 1) — взаємне виключення при доступі до сховища

## Структура проєкту

| Файл | Призначення |
|------|-------------|
| `Program.cs` | Точка входу: введення параметрів, створення потоків |
| `Storage.cs` | Сховище з семафорами: методи `Put()` та `Take()` |
| `Producer.cs` | Логіка виробника: генерація продукції |
| `Consumer.cs` | Логіка споживача: забирання продукції |
| `SharedCounter.cs` | Потокобезпечний лічильник виробленої/спожитої продукції |

## Запуск

```bash
cd ProducerConsumerSharp
dotnet run
```

Програма запитає кількість виробників, споживачів, місткість сховища та загальну кількість продукції.
